## Basic Javascript Syntax

1. Define Variable - we use variable to store things
    ````
    let x = 5;
    let title = "Natual Selection";
    let y = x + 3;
    ````
1. Comment
  - Online comment: //
    ````
    // This is a commit
    ````
  - Multiline comment: /* */
    ````
    /*
     *  I have a lot to say
     */ 
    ````
1. Print something to console
    ````
    let x = 5;
    console.log(x)
    ````
