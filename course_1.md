## Basic Javascript Syntax

1. Basic Javascript statement Syntax
    ````
    let x = 5;
    let answer = "PSC is good";
    ````
1. Comment
  - Online comment: //
    ````
    // This is a commit
    ````
  - Multiline comment: /* */
    ````
    /*
     *  I have a lot to say
     */ 
    ````
1. Print something to console
    ````
    let x = 5;
    console.log(x)
    ````
